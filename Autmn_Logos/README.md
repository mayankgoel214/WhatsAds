## Brand Specifications

- **Primary font**: Manrope (Bold 700 for "autmn", Light 300 for ".ai")
- **Primary color**: #111111 (near-black)
- **Secondary color**: #999999 (gray, for ".ai" subtle variant)
- **Background**: #FFFFFF (white) or #0A0A0A (near-black)
- **Minimum clear space**: Height of the icon on all sides
- **Minimum size**: 80px wide for horizontal, 60px wide for icon only
